# 🎯 AI Resume - JD Matcher

An intelligent resume screening tool that compares resumes with job descriptions and generates a match score using NLP techniques.

## 📌 About the Project

This tool helps job seekers understand how well their resume matches a job description. It highlights missing keywords and gives a similarity score so candidates can improve their resume before applying.

## ✨ Features

- Upload resume in PDF format
- Paste any job description
- Get match score in percentage
- See missing keywords instantly
- Simple and clean web interface

## 🛠️ Tech Stack

- **Python** — core programming language
- **Flask** — web framework
- **Scikit-learn** — TF-IDF vectorization and cosine similarity
- **Pandas and NumPy** — data processing
- **PyPDF2** — PDF text extraction
- **HTML and CSS** — frontend interface

## 🚀 How to Run

1. Clone the repository
   git clone https://github.com/reshmitha345/resume-jd-matcher.git

2. Install dependencies
   pip install flask scikit-learn pandas numpy PyPDF2

3. Run the app
   python app.py

4. Open browser and go to
   http://127.0.0.1:3000

## 📸 How it Works

1. Upload your resume PDF
2. Paste the job description
3. Click Check Match Score
4. See your match percentage and missing keywords
